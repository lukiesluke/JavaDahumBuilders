# JavaDahumBuilders


### Building release

For Dahum Builders
```sh
$ cd ROOT_FOLDER
$ ./gradlew clean :app:assembleRelease
```

For TP Corp
```sh
$ cd ROOT_FOLDER
$ ./gradlew clean :tprealty:assembleRelease
```

Refresh dependencies
```sh
$ cd ROOT_FOLDER
$ ./gradlew clean --refresh-dependencies buildNeeded
```
